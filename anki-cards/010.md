В чем разница между Function-Based Views (FBV) и Class-Based Views (CBV) в Django?

%

FBV - это обычные Python функции, которые принимают request и возвращают HttpResponse. CBV - это классы, которые наследуются от базовых классов Django и предоставляют больше функциональности из коробки (обработка GET/POST, миксины, аутентификация и т.д.). CBV более переиспользуемы, но менее явные.

```python
# FBV
def my_view(request):
    return HttpResponse('Hello')

# CBV
class MyView(View):
    def get(self, request):
        return HttpResponse('Hello')
```
