Как создать URL-маршрут в Django, который подключит view функцию `home_view` из `exersized.views` к корневому пути `/`?

%

```python
# config/urls.py или exersized/urls.py
from django.urls import path
from exersized.views import home_view

urlpatterns = [
    path('', home_view, name='home'),
]
```
