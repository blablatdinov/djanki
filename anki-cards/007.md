Как создать Django модель с полями `title` (CharField, max_length=200), `content` (TextField) и `created_at` (DateTimeField, auto_now_add=True)?

%

```python
from django.db import models

class Post(models.Model):
    title = models.CharField(max_length=200)
    content = models.TextField()
    created_at = models.DateTimeField(auto_now_add=True)
    
    def __str__(self):
        return self.title
```
